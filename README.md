# jekyll-time-to-read-in-polish

:hammer: Display time to read (in polish language)

## Install

1. Copy `_includes/read-time.html` to your project.
2. That's it! :smile:

## Usage

1. On post page add
    
    ```
    {% include read-time.html content=post.content %}
    ```
2. That's it! :smile:
